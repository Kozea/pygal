## Description

**pygal** is a dynamic SVG charting library written in python.
All the documentation is on [www.pygal.org](http://www.pygal.org)


## Installation

As simple as:

```
    $ pip install pygal
```



## Test

Pygal is tested with py.test:


```
    $ pip install pytest
    $ py.test
```
